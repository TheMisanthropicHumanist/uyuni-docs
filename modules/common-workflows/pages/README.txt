Follow the instructions given here each time you create a new document for the workflow you are describing.

1. Make a copy of the workflow-TEMPLATE.adoc file and rename it to workflow-your-chosen-filename.adoc

2. Use full words in "your-chosen-filename", separting each word with the dash "-". 
E.g. instead of workflow-clm.adoc, use workflow-content-lifecycle-management.adoc.

3. The instructions within the template are not strict rules but guidelines for creating the document.
They outline main types of information the file needs to contain.

4. When the workflow file is ready, add the new topic to the navigation bar. 
All topics are listed in alphabetical order.

5. Add entry to .changelog file.

6. Submit the PR.
