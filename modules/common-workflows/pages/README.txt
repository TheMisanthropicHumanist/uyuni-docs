Follow the instructions given here each time you create a new document with the workflow you want to describe.

1. Make a copy of the workflow-template.adoc file and rename it as workflow-your-chosen-filename.adoc

2. Use full words instead of "your-chosen-filename", separateing each word with the dash "-". E.g. instead of workflow-clm.adoc, use workflow-content-lifecycle-management.adoc.

3. Follow the instructions within the template to create the document content.

4. When the workflow file is ready, make sure you have added the topic you described to the navigation bar. All topics are listed in alphabetical order, please add yours at the correct place.

5. Don't forget to add entry to .changelog file.
